# Persistent Design Rules

- Preserve the bold glassmorphism direction with high-contrast text on a deep blue-green base.
- Keep accent colors anchored to `--coral`, `--teal`, and `--gold`; avoid introducing purple-led palettes.
- Use `Space Grotesk` for expressive headings and `Sora` for body copy unless the user asks for a different type system.
- Prefer layered gradients, blurred ambient shapes, and translucent cards over flat sections.
- Maintain motion as part of the experience: scroll reveals, soft floating movement, tactile hover states, and restrained transitions.
- Keep project layouts in a Bento Grid and preserve click-to-expand behavior for case-study depth.
- Treat the portrait as a premium hero asset with soft framing, not a small avatar.
- Reuse root CSS tokens before adding new colors, shadows, spacing scales, or radii.
- Keep the portfolio one-page and responsive first, with mobile layouts feeling intentional rather than reduced.
